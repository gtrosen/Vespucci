1 Vespucci
2 ========
3 Vespucci is a free/libre/open-source, cross-platform tool for spectroscopic
4 imaging

7 Binary Releases:
8 Because the members of my group primarily use Windows machines, most releases
9 will only include Windows binaries. Major releases may include binaries for
10 Windows, Mac OSX, and GNU/Linux.

13 Compiling Vespucci:
14 ===================
15 Compiling Vespucci from source is easy on Unix-like systems. Assuming you
16 have the proper pre-requisites (and Vespucci.pro is properly edited to reflect
17 their paths):
18 
19 qmake && make && make install
20 
21 Should do the trick. This does not require elevation as the default install path
22 is $HOME/Vespucci. You can modify INSTALL to install for all users if you'd
23 like.
24 
25 Pre-requisites:
26 If you can compile and install MLPACK and have the Qt framework installed,
27 then you can compile and install Vespucci. You should have the following
28 libraries on your system:
29 
30 Qt
31 MLPACK
32 ARMADILLO (compiled shared library)
33 LAPACK and BLAS (or high speed replacement like OpenBLAS, Accelerate, MKL, etc.)
34 ARPACK (or ARPACK-ng)
35 Boost (program_options, unit_test_framework, random, and math (c99))
36 LibXML2
37 QCustomPlot (compiled shared library)
38 
39 All of the above packages, with the exception of MLPACK and QCustomPlot, are
40 readily available from most major GNU/Linux repositories. Installation on Mac
41 OS is similarly easy, though some packages may need to be compiled and installed
42 manually.
43 
44 The included Qt profile uses the library paths used by Ubuntu's package manager.
45 Frameworks are used for Mac OS when possible.
46 
47 For Unix-like systems, shared libraries are the default. You will need to
48 compile and install static versions of these libraries if you want a stand-
49 alone executable.
